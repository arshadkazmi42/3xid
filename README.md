# 3xid

[![Build](https://img.shields.io/travis/com/arshadkazmi42/3xid.svg)](https://travis-ci.com/arshadkazmi42/3xid/)

xxx-xxx-xxx unique id generator

## Install

```
npm i 3xid
```

## Usage

```javascript

const xId = require('3xid');

console.log(xId.generate());

// OUTPUT
// a123-v432-c442

```
